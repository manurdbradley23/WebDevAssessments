HTML & CSS Quick-Reference Guide

Reference for HTML and CSS properties, with examples of their usage.

HTML Tags

  • <h1> to <h6> - Defines different levels of headings.
    • <h1>This is a heading</h1>
  • <p> - Defines a paragraph of text.
    • <p>This is a paragraph.</p>
  • <a> - Defines a hyperlink.
    • <a href="https://example.com">Visit Example</a>
  • <img> - Embeds an image
    • <img src="image.jpg" alt="Description">
  • <li> - For sorting a list
    • <li>100 grams Penne</li>
<li>1 Tomato can</li>
  • <ul>, <ol>, <li> - Defines unordered and ordered lists.
    • <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
  • <div> - Creates a division.
    • <div>
    <h1>title</h1>
    <p>informationinformation</p>
</div>
  • <span> - An incline container for styling
    • <span class="">Some Text</span>
  • <table>, <tr>, <td> - Table, table row and table data
    • <table>
    <tr>
        <td>data1</td>
        <td>data2</td>
        <td>data3</td>
    <tr>
</table>

    CSS Properties

  • color - defines the text color
    • p {
     color: blue;
}
  • background-color - sets the background color
    • body {
    background-color: lightgray;
}
  • font-size - adjusts the size of the text
    • h1 {
    font-size: 24px;
}
  • margin - controls spacing around elements
    • div {
    margin: 20px;
}
  • padding - controls spacing inside elements
    • section {
    padding: 15px;
}
